Navigation

You have already figured out the first step
towards using this program: navigation.

Above and below you are the navigation keys,
which you can use to move from one section
of a lesson to the next.

The left arrow button (<<|) moves you back a section.
The right arrow button (|>>) moves you forward a section.
The Lesson Home button returns you
to where you selected this lesson.
